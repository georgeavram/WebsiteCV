Projects

Delivery Tracking System

This was a team-based project that I was part of as my 2nd year of college with multiple iterations where data can be passed through an MQTT script to append to our database and then be displayed on a secure localhost live website. The main function was to include a website order searching system via username and password where users can check their order status, date, and location based on the database information. Furthermore, Azure was used for project documentation, coordination, and setting assignments/assignment due dates.

Difficulties encountered
  • Creating a proper secure live connection from the database to the website using an MQTT broker was difficult but was handled by studying the functions and test running scripts.
  • Using JavaScript to display database content and to use it in a search function proved difficult but was overcome with enough teamwork and testing code.
IOT Temperature Device

This was my Final year project and involed creating an IOT device out of a Rasperry pi that could pick up the Temperature of an environment then send the data to AWS Services through secure MQTT connection and be able to pull the data to display on a user interface on a seperate device through an MQTT publish while also publishing data for perminant storage on Dynamo DB. My use case for this project is a cheap computer device to collect environmental data with temperature as the base with low bandwidth and power consumption

Difficulties encountered
  • Building the script to collect data from the Raspberry pi and pushing it to my AWS service proved difficult at first as I had to account of collecting data, cleaning and properly sending the data in correct intervals to eachother as incorrect timing proved to push incorrect data for a certain point in time, this was solved just by putting stops in the script till certain data pieces are collected
