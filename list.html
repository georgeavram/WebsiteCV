━━━ Bootstrap Elements ━━━

Below is a list of the main Bootstrap elements I used to create my Main page

  • Responsive Grid System

    • I used the Grid to evenly space out and sometimes center my contents e.g. col-2, col-12 etc.

  • Buttons

    • Bootstrap buttons were used for my grades, the years and my contact information at the bottom of the page

  • parralax-image

    • The parralax-image feature for the background on my profile

  • navbar

    • Using the bootstrap navbar feature helped me better arange my navbar features

  • Collapse

    • Collapse was used to reval and hide my grades in the education section

  • Button

    • I used the default bootstrap button designs for most of my buttons

  • Model

    • The model feature was used to show a pop-up of my contact media to the user

Div class bootstrap features I used
  • Padding and Margin

    • I used elements such as p-4/pt-5 and m-4/mt-5/mx-1 to proved spaceing without using CSS on some elements

  • text-center and justify-content-center

    • These we're used to center content in certain div's

  • Img-fluid

    • this was used to make the images I used reponsive to div's that they were in, this was so images can be re-sized for mobile
